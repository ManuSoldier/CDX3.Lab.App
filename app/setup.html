RIO Setup

Overview

RIO provides a sample application along with the artifacts. The application is located at app folder and the artifacts are available at resources folder. Resources folder includes the all necessary assets related to layout and theme such as css, js, fonts, images.

Run the sample

To run the sample, expand the contents of the zip file and run "npm install" command to download all necessary modules. Then run "npm start" command that opens a browser and runs the homepage.

Core Layout

Core Layout is a set of CSS classes used to create responsive grid layouts with additional utilities. Rio uses Core Layout for demo pages, however for grid layout you may also choose another utility like Grid CSS or Bootstrap Grid.
